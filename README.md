# DALALI MTANDAONI

DALALI MTANDAONI is a Django-based online marketplace that connects property owners and seekers.

## Features

* Custom User Authentication
* Owner and Seeker Accounts
* Property Categories
* Property Listings
* Multiple Image Uploads
* Admin Approval System
* Property Management Dashboard

## Categories

* Houses for Rent
* Rooms for Rent
* Plots for Sale
* Cars for Hire

## Technologies

* Django
* Bootstrap
* SQLite (Development)
* Pillow
